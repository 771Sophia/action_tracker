PWA 安裝與小工具模式指引

📱 在電話（手機）端安裝

1. 啟動本程式的本地伺服器後，確保您的電話與電腦處於同一個 Wi-Fi 網絡。

2. 在電話瀏覽器（iOS 的 Safari 或 Android 的 Chrome）輸入您的 電腦 IP 地址:8000。

3. 點擊分享或選單按鈕，選擇 「新增至主畫面」 (Add to Home Screen) 或 「安裝 App」。

4. 手機案頭即會出現 App 圖示，點擊後會以無網址列的**獨立 App 模式**運行，完美離線使用！

💻 在 Windows 下方工作列（BAR）彈出

1. 在 Windows 電腦 Chrome 或 Edge 瀏覽器開啟網址後，在網址列右側點擊 「安裝此應用程式」 (Install App) 圖示。

2. 安裝完成後，右鍵點擊工作列中本程式的圖示，選擇 「固定至工作列」 (Pin to Taskbar)。

3. 點擊本程式頂部的 「極簡小工具模式」 按鈕，將視窗縮小調整，即可當作隨手點擊的浮動小工具！
